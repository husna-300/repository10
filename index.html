Steps of using Git and GitHub

An intro to git and github for beginners(tutorial)

  1. Install Git and creat a GitHub account ( git --version , is to check if git is properly installed)
  2. git --version
  3. git config --global user.name " "
  4. git config -- global user .email""
  5. Initialize git ( git init ) you just creat your first git repository
  6. Then check the Git Status and see if it is a part of our repo
  7. git add ( the file should be staged.)
  8. then check the status
  9. Git commit -m " first release of Hello World"
  10. git branch -M main
  • now creat repositor
  • git remote and origin URL
  • git push -- set- upstream origin master
